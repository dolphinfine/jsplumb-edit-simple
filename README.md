# jsplumb简易流程图编辑器

![image](http://note.youdao.com/noteshare?id=79a501230a550b98ab0f5b70347a8b30)

## 实现了以下功能

- 从左侧拖拽流程到画布
- 画布节点内容编辑
- 画布节点保存、清空、还原
- 流程图的放大、缩小、平移
- 节点删除、连线删除
- 节点复制粘贴

### 安装依赖
```
npm i
```

### 运行
```
npm run serve
```
